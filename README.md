Removes ads on Yandex services on web.

## How to Use
How to apply the blocking.

1. **Install Stylus:**
   - [Chromium Browsers](https://chromewebstore.google.com/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)
   - [Gecko Browsers](https://addons.mozilla.org/en-US/firefox/addon/styl-us/)
2. **Add Style**
   Add the style by going to [this link](https://userstyles.world/style/24888/noyaads) and clicking "Install", then "Install Style".
3. **Enjoy!**
4. **Report Missing Apps**

   If you use a Yandex service/app that doesn't have its ads blocked by this style, [create a GitHub Issue](https://github.com/asboy2035/NoYaads/issues/new).


## Supported Services
- Image Search
- 360 Mail/Calendar
- Weather
